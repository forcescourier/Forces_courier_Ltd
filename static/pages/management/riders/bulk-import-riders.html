Import Instructions

How to prepare your file:
  • Use CSV or Excel format (.csv, .xlsx, .xls)
  • First row should contain column headers
  • Required fields: Full Name, Phone, Email, National ID, Vehicle Type
  • Optional fields: Address, Emergency Contact, License Number, Zone Assignment
  • Maximum file size: 10MB
  • Maximum records per import: 200
Required Fields:
  • Full Name: Rider's complete name
  • Phone: Contact phone number
  • Email: Valid email address
  • National ID: Government ID number
  • Vehicle Type: Bike/Scooter/Car
Optional Fields:
  • Address: Home address
  • Emergency Contact: Emergency phone number
  • License Number: Driving license number
  • Zone Assignment: North/South/East/West
